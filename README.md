## How I work (my SEO operating system)?

### Technical SEO
- Crawl & indexation fixes: robots, sitemaps, canonicals, pagination, parameter handling
- Site architecture: clean templates, intent-based URL structures, internal linking systems
- Performance: CWV priorities, image strategy, caching/CDN (Cloudflare), on-page hygiene

### Content & Growth
- Programmatic SEO (pSEO): scalable templates that still feel human and intent-complete

### GEO (AI Search Optimization)

- Fact-first structure: avoid fluff, increase precision, improve answer completeness
